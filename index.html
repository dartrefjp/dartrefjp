このサイトについて

GoogleがJavaScriptの置き換えを目指し開発を進めた、ウェブプログラミング言語Dartの日本語リファレンスサイトです。

日本語資料が少ないので作りました。言語仕様やAPIリファレンス、ニュースなどの日本語での情報を提供していく予定です。
現状、まだまだ内容が不足しているので協力して頂けると嬉しいです。

（※このサイトの情報は Dart : Structured web programming や関連した情報を和訳・編集した内容も含みます。
それらの最新の情報を確認したい場合は Dartリソースのリンク集 - 日本語以外 が参考になるでしょう。）
